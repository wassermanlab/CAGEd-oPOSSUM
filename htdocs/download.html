The FANTOM5-oPOSSUM source code is also available as a public repository on GitHub under wassermanlab/FANTOM5-oPOSSUM.

The FANTOM5-oPOSSUM databases may be downloaded directly from our database servers using the mysldump facility.

There are currently two species databases:

FANTOM5_oPOSSUM_human
FANTOM5_oPOSSUM_mouse

The databases are hosted on fantom.cmmt.ubc.ca and may be accessed using MySQL username 'fantom_r'. It may be necessary to use the --skip-lock-tables option to mysqldump, e.g.:
$ mysqldump -h fantom.cmmt.ubc.ca -u fantom_r --skip-lock-tables FANTOM5_oPOSSUM_human > FANTOM5_oPOSSUM_human.sql
